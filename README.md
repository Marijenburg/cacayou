# Cacayou 🐊

Un petit jeu d'exploration fait à la main par **Elaijah** (et son papa Charlie).

Balade-toi dans une forêt pastel : ramasse ta hache, coupe des arbres, récolte des
bûches, croise des monstres qui errent. Tous les personnages, décors et sons sont
dessinés et enregistrés par Elaijah et Charlie ; le code se contente d'assembler et
d'animer leurs créations.

## Jouer

Ouvre `index.html` (ou l'URL déployée) sur téléphone ou ordinateur.
- **Se déplacer** : touche/clique un point de l'écran (le perso y va en contournant les obstacles), ou clavier WASD / ZQSD / flèches.
- **Couper un arbre** : ramasse la hache (près du départ), puis touche un arbre.
- **Ramasser** : passe sur les bûches tombées.
- Bouton plein écran en haut à droite ; l'écran s'adapte en portrait ou paysage.

## Technique

Vanilla JS + Canvas 2D, **zéro dépendance**. Fichiers statiques : `index.html`,
`game.js`, `assets/`. Déployable tel quel (Cloudflare Pages, itch.io, n'importe quel
hébergement statique).
